linux内核

1，硬件操作

2，系统调用，把对硬件的操作封装成一个个系统调用，相当于函数

3，终端命令，用来触发系统调用，相当于程序

Linux目录

1，/表示根目录，Linux只有一个盘

  • bin：存放可执行文件
  • etc：存放配置文件
  • home：每个用户的家文件夹

2，操作应该在home的用户文件夹里

Linux命令

1，放大终端

  • ctr+shift+=：shif+=就等于+号，原理就是ctr加+号

2，缩小终端

  • ctr+-：c加减号

3，基本命令

  • cd
  • ls
  • pwd
  • mkdir
  • touch
  • rm
  • clear
  • xx –help 查看xx命令帮助信息
  • man xx，查看xx指令的操作手册

文件相关命令

ls

1，作用：列出目录下的文件和目录

2，使用

  • ls -a：包含隐藏文件，隐藏文件是.开头的文件
  • ls -l：列出文件的详细信息
  • ls -h：配合 -l使用，让大文件用k或m显示，而不是只用字节显示
  • ls -lha:可以合在一起

3，通配符

  • ？代表任意一个字符
  • *代表任意多个字符，可以是0个
  • [1,2,3,5] 代表一个字符，只能是指定这几个中的一个
  • [1-4] 等同于[1,2,3,4]，代表一个字符

cd change direction

1，作用：切换目录

2，使用

  • cd/cd ~ 代表回到用户的家目录，~直接代表家目录，cd ~/zxr
  • cd xxx代表去xxx目录
  • cd .代表当前目录
  • cd ..代表上级目录
  • cd -代表在最近两个目录直接来回切换

创建和删除

1，touch ~/zxr.txt代表在用户的家目录创建一个zxr.txt文件，如果该文件存在，那么会该表该文件的修改时间

2，mkdir ~/zxr代表在家目录创建一个zxr的目录，目录存在，或者目录与文件同名则创建失败

  • mkdir -p ~/zxr/friends表示在家目录创建zxr，在zxr目录再创建friends目录

3，rm a.txt代表删除文件，不能删除目录

  • rm -f代表强制删除，不会提示错误信息
  • rm -r a代表递归删除a目录以及目录以下的文件
  • rm可以使用通配符：rm -rf *，代表删除目录下所有的文件和目录

文件拷贝和移动

1，cp 源文件(路径加文件名) 目标文件(路径加文件名)，代表把源文件复制为目标目录下的新文件

  • cp -i ，在目标目录存在与目标文件名同名的文件时，提示是否要覆盖，y or n
  • cp -r，复制当前目录为新的目录，包含目录底下的目录和文件

2，mv 源文件 目标文件，移动文件并重命名为目标文件

  • mv 源文件 目标文件，如果目标文件就是源文件，则可以重命名源文件为目标文件

3，tree xx，显示xx目录下面的目录文件树

  • tree -d xx，只显示目录

文件查看

1，cat xx.txt代表查看文件内容

  • cat -b 代表显示行号，空行没有行号
  • cat -m代表显示行号，空行也有行号

2，more xx.txt 代表分页查看文件内容

  • 按f或空格查看下一页
  • 按b查看上一页
  • 回车查看下一行
  • q退出

3，less xx.txt是more的增强版

  • 空格键：向下翻一页
  • b 键：向上翻一页
  • 上/下箭头键：向上/下滚动一行
  • g 键：跳到文件开头
  • G 键：跳到文件末尾
  • q 键：退出 less
  • /zxr搜索zxr字符并高亮
  • esc + U，取消高亮

4，grep zxr zxr.txt表示在zxr.txt搜索zxr，显示包含zxr的行

  • grep -n zxr zxr.txt显示包含行号
  • grep -v zxr zxr.txt显示不包含zxr的行号
  • -i表示忽略大小写
  • 使用通配符
    • grep -n ^hf zxrs.txt 查询hf开头的行并显示行号
    • grep -n fh$ zxrs.txt 查询fh结尾的行并且显示行号

echo和重定向

1，echo daniuma执行后会在终端显示daniuma

2，echo daniuma > a.txt 代表把终端展示的内容输出到a.txt，覆盖原有内容

2，>> 追加内容，而不是覆盖

管道

1，把左边命令的输出作为右边命令的参数，相当于在管子左边放东西，从管子右边取出来作为参数

2，ls -lh | grep zxr代表把ls的输出作为grep的参数，grep会筛选出包含zxr的行

远程管理相关命令

关机重启

1，shutdown time：在time时刻关机，也可以写shutdown +10，表示十分钟后关机，shutdown now···

2，shutdown -r time表示重启

3，shutdown -c表示取消重启或者取消关机

查看ip和ping命令

1，查看ip地址

  • ifconfig(interface config)，ess开头代表物理网卡，inet开头代表ipv4地址
  • ip addr show，一样看ess网卡的inet

ssh远程连接

1，ssh是一个远程连接协议，对指令和数据的网络传输途中进行了加密

2，在xshell中使用ssh -p 端口 用户名@ip地址，然后输入密码，就可以连接了

  • 端口默认是22，取决于服务器安装的sshserver的进程端口
